TRIANGLE SUM[PATTERN]

  • Step 1: Read an integer n from the user.
  • Step 2: Iterate i from 0 to n (inclusive).
  • Step 3: This controls the number of rows to be printed.
  • Step 4: For each i, iterate j from n-i down to 0 (inclusive).
  • Step 5: Print spaces (" ") to align the pattern correctly.
  • Step 6: For each i, iterate k from 1 to i (inclusive),Print stars ("* ").
  • Step 7: After printing the stars for the current row (i), move to the next line (cout << endl;).
  • Step 8: Repeat steps 2-5 until all rows are printed.
  • Step 9: Display the pattern as specified.




HOLLOW SQUARE[PATTERN]

  • Step 1: Read an integer n from the user.
  • Step 2: Loop through each row i from 1 to n.
  • Step 3:  Loop through each column j from 1 to n.
  • Step 4: If i is 1 or i is n, or j is 1 or j is n, print "*" (border).
  • Step 5: Otherwise, print " " (interior).
  • Step 6: Move to the next line after printing each row.
  • Step 7: Display the rectangular pattern with borders made of "*".




ARRAY LIST

  • Step 1:  Read the integer n, representing the size of the array.
  • Step 2: Read the array arr of size n.
  • Step 3:  Initialize maxSum to a very small number (or the value of the first element arr[0]).
  • Step 4: Initialize sum to 0.
  • Step 5: Traverse each element arr[i] from 0 to n-1.
  • Step 6: Update sum by adding arr[i] to it.
  • Step 7: Update maxSum to be the maximum of maxSum and sum.
  • Step 8: If sum becomes negative, reset sum to 0 (start a new potential subarray).
  • Step 9: maxSum is the maximum subarray sum found.




INTERCHANGE ROW

  • Step 1: Initialize Matrix: Create a 4x4 matrix A.
  • Step 2: Read the array arr of size n.
  • Step 3: Input Matrix Elements: Read 16 integers from input to populate matrix A.
  • Step 4:  Swap Rows: Swap the elements of the first row with the elements of the last row.
  • Step 5: For each column index j from 0 to 3.
  • Step 6: Swap A[0][j] with A[3][j].
  • Step 7: Print the matrix A after swapping.
